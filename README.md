## Features

- Full ghost mode (flexible)
- Messages history
- Anti-recall
- Keep locally (in group delete menu)
- Save to self (in spam report menu)
- Font customization
- Streamer mode
- Local Telegram Premium
- Translator
- Media preview & quick reaction on force click (macOS)
- Enhanced appearance

And many more.

## Downloads

### Windows

#### Official

You can download prebuilt Windows binary from [Releases tab](https://github.com/greenyarik0505-jpg/FluxGramDesktop/releases).

#### Winget

```bash
winget install FluxGramDesktop
```

#### Scoop

```bash
scoop bucket add extras
scoop install fluxgram
```

#### Self-built

Follow [official guide](https://github.com/greenyarik0505-jpg/FluxGramDesktop/blob/dev/docs/building-win-x64.md) if you want to build by yourself.

### macOS

#### Official

You can download prebuilt macOS package from [Releases tab](https://github.com/greenyarik0505-jpg/FluxGramDesktop/releases).

### Arch Linux

#### From source (recommended)

Install `fluxgram-desktop` from AUR.

#### Prebuilt binaries

Install `fluxgram-desktop-bin` from AUR.

### NixOS

Install `fluxgram-desktop` from nixpkgs.

### Remarks for Windows

Make sure you have these components installed with VS Build Tools:

- C++ MFC latest (x86 & x64)
- C++ ATL latest (x86 & x64)
- latest Windows 11 SDK
